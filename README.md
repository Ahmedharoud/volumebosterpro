# Volume Booster Pro

Boost browser audio up to 600%.

## Features
- Volume control up to 600%
- Works on YouTube, Netflix, Spotify
- Smooth slider UI
- Preset buttons
- Dark mode design

## Installation

1. Download the repository
2. Open Chrome Extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select extension folder
